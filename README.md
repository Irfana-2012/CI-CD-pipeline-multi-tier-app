# CI-CD-pipeline-multi-tier-app

Designed and implemented a fully automated CI/CD pipeline to build , test and deploy web application
Implemented Blue/Green deployment for zero downtime
Built a complete pipeline using AWS CodePipeline for orchestration
Integrated AWS CodeBuild to automate build and testing phases
Deployed applications using AWS CodeDeploy with zero downtime deployment strategy
Implemented IaC using CloudFormation
Artifacts are stored securely in S3 bucket
Configured IAM Roles and policies for secure service interactions
